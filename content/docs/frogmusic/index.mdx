# 簡介

## 想要使用音樂機器人，但是又覺得功能太少嗎？
我們提供了強大的點播系統，而此系統為本機器人的亮點。由 Lavamusic 所改寫，並且製作成一個高音質的 Discord 機器人。

## 初次使用？
記得設定 [點播頻道](/docs/frogmusic/quicksetup/setup) 喔~

## 為什麼 Discord 封鎖 YT 播放呢？
Discord TOS 提及了很多事情，因為 YT 相較於其他播放軟體大很多，又加上他們有自己的 YT Music 以及廣告功能，所以才會不允許以這樣的方式在 Discord 中來播放。

## 功能亮點
我們製作了超讚的點播系統，供使用者自行點播，並且可以配合不同來源以及 URL。預設搜尋引擎為 Spotify ，這樣我們的播放系統，能讓使用者們聽到超級震撼或超好聽的歌曲！
